#### Purpose
To reduce the communication gap of student with complete anonymity. 

#### So whats the possible solution??
CONFAB is a platform where a person of specific organization can login through the email account provided by that organization, which will lead to an open question and answer platform of that specific organization. User can ask question on that forum anonymously, only his/her designation or post will be displayed so that a user can ask question freely knowing that his/her identity will not be revealed.
